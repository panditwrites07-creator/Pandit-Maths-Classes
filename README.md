# Pandit Maths Classes Website

A professional website for Pandit Maths Classes, a coaching center specializing in mathematics education for classes 5-10 with focus on entrance exams like Sainik School and Navodaya School.

## Features

- **Responsive Design**: Works perfectly on mobile, tablet, and desktop devices
- **Modern UI**: Clean, professional design with gradient backgrounds and cards
- **Complete Information**: All necessary details including contact information, address, classes offered
- **Success Stories**: Showcases the 3-4 students who have already joined this year
- **Call-to-Action**: Clear enrollment prompts and contact options

## Website Sections

1. **Header**: Logo, name, and contact information prominently displayed
2. **Hero Section**: Main introduction with call-to-action buttons
3. **Features**: Why choose Pandit Maths Classes (small batches, exam focus, etc.)
4. **About**: Information about the coaching and classes offered (5th to 10th)
5. **Success Stories**: Testimonials from current students
6. **Location**: Address details (Behind Sneha Petrol Pump)
7. **Enrollment Banner**: Urgent call for new admissions
8. **Contact Section**: Multiple contact options (phone, email, address)
9. **Footer**: Copyright and summary information

## Technical Details

- Built with React + TypeScript + Vite
- Styled with Tailwind CSS
- Uses Lucide React for icons
- Fully responsive design
- SEO optimized with proper meta tags

## Development

To run the project locally:

```bash
npm install
npm run dev
```

To build for production:

```bash
npm run build
```

## Deployment

The built files are in the `dist` folder and can be deployed to any static hosting service like Netlify, Vercel, GitHub Pages, or a traditional web server.
